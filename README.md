# Kaartje Analytics Dashboard

Live analytics for the Kaartje app (TestFlight). The page fetches data
directly from a Google Apps Script endpoint — no build step, no backend.

Hosted with GitHub Pages: the site is just `index.html`.

**To update:** copy the latest `CardKeep/dashboard.html` from the project
over `index.html` and commit (or re-upload via the GitHub web UI).
